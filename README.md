## Smart Waste Exchange System

## Problem
Eswatini faces inefficient waste collection, lack of reporting systems, and poor visibility of environmental pollution hotspots.

## Solution
A civic AI-powered waste management platform that connects citizens, recyclers, and administrators.

## Features
- Public waste reporting (no login required)
- Automated recycler assignment
- Role-based dashboard (Recycler + Admin)
- Waste lifecycle tracking
- AI-based hotspot detection
- Interactive map visualization

## Tech Stack
- Streamlit
- Pandas
- Folium
- Python

## Workflow
Reported → Assigned → Accepted → Collected → Verified

## Impact
- Reduces illegal dumping
- Improves recycling efficiency
- Enables data-driven environmental planning

## Future AI Expansion
- Predict waste hotspots using ML
- Optimize recycler routes
- Real-time SMS alerts
